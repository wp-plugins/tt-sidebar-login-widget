== Description ==

*I Appreciate if you please give reviews and any suggestions after using this plugin.*

TT Sidebar Login is a widget plugin developed by Rashed Latif. There are many similar plugins available that Can be used for logging into Wordpress sites. This plugin can be downloaded [here]. It is styled to fit into different themes but still there could be some problem in styling depending on the themes which can be fixed by anyone who knows CSS and HTML. Version 1.0 includes


* Login form with "Register" and "Forgot password link"
* Displays avatar when logged in
* Displays member's role and total number of posts by the user when logged in
* Displays "Dashboard" link when logged in
* Shows "Profile" link when the user is logged in
* Widget option have the ability to turn on/off "Remember me" check box, "Register", "Forgot password", "Dashboard", "Profile" links and "Avatar" option.


Author will not provide any support for this plugin.  Other developers can use the codes of this plugin as reference and can use it in their codes.


== Installation ==

* Step 1: Upload plugin to the `/wp-content/plugins/` directory
* Step 2: Activate the plugin through the 'Plugins' menu in WordPress
* Step 3: The activated widget will be available to widget area. Select Appearence > Widget.
* Step 4: Drag the TT Sidebar Login widget to side bar. It will be displayed on your website. By default "Remember me" option, "Register", "Forgot Password" link on login form and "Avatar", "Dashboard" and "Profile" link on logged in screen is activated. If you want to disable any one of the options it can be done from widget option. It's very simple, just a matter of selecting and unselecting an option.
